## O que é o Power BI

Power BI é uma ferramenta de Business Intelligence (BI) desenvolvida pela
Microsoft, usada para conectar dados de diferentes origens — planilhas,
bancos de dados, sistemas de CRM, entre outros —, transformá-los e
apresentá-los em dashboards visuais e interativos.

Em vez de olhar números soltos em uma planilha, o Power BI permite
enxergar tendências, comparações e padrões de forma visual, facilmente
compartilhável com o time.

## Os três componentes principais

- **Power BI Desktop**: aplicativo gratuito para Windows, onde os
  relatórios são criados e os dados são conectados e modelados
- **Power BI Service**: a versão em nuvem, onde os relatórios são
  publicados e compartilhados com outras pessoas da empresa
- **Power BI Mobile**: aplicativo para acompanhar dashboards pelo celular

## Como o Power BI transforma dados em dashboards

O fluxo típico de trabalho no Power BI segue três etapas:

1. **Conectar**: importar dados de uma ou várias fontes (Excel, bancos de
   dados, serviços web)
2. **Transformar**: limpar e organizar os dados usando o Power Query,
   removendo inconsistências e ajustando formatos
3. **Visualizar**: criar gráficos, tabelas e indicadores que respondem às
   perguntas de negócio, organizados em um ou mais dashboards

## O que é DAX e quando você vai precisar dele

DAX (Data Analysis Expressions) é a linguagem de fórmulas do Power BI,
parecida em espírito com fórmulas do Excel, mas pensada para trabalhar com
grandes volumes de dados relacionados entre tabelas. Você precisa de DAX
quando quer criar métricas mais sofisticadas — por exemplo, comparar
vendas do mês atual com o mesmo mês do ano anterior.

Para dashboards simples, muitas vezes o Power BI resolve sem nenhuma
fórmula DAX, só conectando e organizando os dados visualmente.

## Por que empresas adotam o Power BI

- Substitui relatórios manuais atualizados "na unha" todo mês
- Permite que decisões sejam tomadas com base em dados atualizados, não
  em relatórios defasados
- Centraliza informações que antes ficavam espalhadas em planilhas
  diferentes, cada uma com uma versão "da verdade"
- Tem uma curva de aprendizado inicial relativamente rápida comparada a
  outras ferramentas de BI mais complexas
