## O que é o Bizagi Modeler

O Bizagi Modeler é um software de modelagem de processos que segue o
padrão BPMN. Ele permite desenhar diagramas de forma visual, com um
sistema de arrastar e soltar elementos como tarefas, eventos e gateways,
sem precisar programar nada.

É uma das ferramentas mais recomendadas para quem está começando a
trabalhar com BPMN, justamente por ser gratuita para modelagem e ter uma
interface simples de aprender.

## Principais recursos do Bizagi Modeler

- **Modelagem visual em BPMN**, com todos os elementos padrão disponíveis
  na barra de ferramentas
- **Colaboração**: vários usuários podem trabalhar no mesmo diagrama,
  dependendo do plano
- **Publicação em nuvem**, permitindo compartilhar o diagrama com um link,
  sem precisar exportar arquivos
- **Documentação automática**: o Bizagi consegue gerar documentação do
  processo a partir do próprio diagrama, economizando um trabalho manual
  que costuma ser chato de fazer

## Bizagi Modeler vs Bizagi Automation

Vale entender essa diferença antes de pesquisar mais sobre a Bizagi:

- **Bizagi Modeler**: ferramenta de modelagem, gratuita para uso
  individual — é sobre isso que este artigo trata
- **Bizagi Automation (ou Bizagi Engine)**: plataforma paga que permite
  transformar o processo modelado em um sistema executável, com
  formulários, regras e integração com outros sistemas

Ou seja: você pode usar o Modeler gratuitamente para desenhar e entender
processos, e só considerar a Automation quando quiser realmente
automatizar a execução deles.

## Quando faz sentido usar o Bizagi

O Bizagi Modeler é uma boa escolha quando:

- Você quer desenhar processos seguindo rigorosamente o padrão BPMN
- Sua equipe precisa colaborar em tempo real nos diagramas
- Você quer gerar documentação de processos sem trabalho manual extra

Para diagramas muito simples ou uso ocasional, ferramentas genéricas de
diagramação também resolvem — mas o Bizagi se destaca quando BPMN é
levado a sério como prática contínua da empresa.
